# Rubik’s cube photo solver

take 6 photos of your cube, fix colors if needed, get moves to solve it.

## Setup

1. Install [Node.js](https://nodejs.org/) (LTS is fine).
2. In the project folder, run:

```bash
npm install
```

3. Start the app:

```bash
npm run dev
```

4. Open **http://localhost:3000** in your browser.
